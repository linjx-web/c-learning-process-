# 第11课 特殊运算符

在用C++编程时，常用到的特殊运算符有括号( )，下标[ ]，成员运算符( - > , . )等几种。

## 一、小括号运算符

数学口诀“先乘除后加减”。

C++语言中，求余运算也是除法，求余运算和除法运算优先级相同。为了改变运算优先级，用小括号( )来实现。

例如：((1+2)*3-(5-4))/2=(3*3-1)/2=4

数学上有中括号[ ]和大括号{ }的概念，但在C++中这两个括号另有用途。

## 二、逗号运算符

逗号运算符用来将若干表达式连接起来，一般格式为：表达式1,表达式2,表达式3,……,表达式n

例如：3+5,6+8

“逗号表达式” 又称为 “顺序求值运算符”。

逗号表达式的求解过程是：先求解表达式1，再求解表达式2，整个逗号表达式的值是表达式2的值。


# 第12课 标准数据类型范围及转换

## 一、内存与数据类型

内存又称为主存，存储的信息可以被CPU直接访问，这种存储原理是由冯·诺伊曼提出的计算机的基本原理。

内存中有很多的存储单元，每个单元可以存放1个8位的二进制数，即1个字节（Byte，简称“B”）。

内存中的每个字节各有一个固定的编号，这个编号称为地址，CPU在存取存储器中的数据时，是按地址进行的。

所谓存储器容量指的是存储器中所包含的字节数，通常用KB、MB、GB、TB和PB作为存储器容量单位，之间的关系如下：

1KB = 1024B ; 1MB = 1024KB ; 1GB = 1024MB ; 1TB = 1024GB ; 1PB = 1024TB ;

## 二、整型的溢出

整数在内存中能存储的最大值和最小值是固定的，当我们要存储的数大于这个固定的最大值时，就会导致整数的溢出。

```cpp
int x=2147483647;
cout<<x+1;
```

输出结果不是2147483648，而是-2147483648，因为发生了溢出。

## 三、混合运算时的类型转换规则

整型、实型、字符型数据间可以进行混合运算，为了保证运算精度，系统在运算时的转换规则是将存储长度较短的运算类型转成存储长度较长的类型，然后再进行处理。

char,short int → int → unsigned int →long int → double ←float

说明：

（1）char,short int → int  和 double ←float中表示必定会进行的转换，如float型数据必先转换成double型数据，然后与其他操作数进行运算。与此类似，char型或short型数据必先转换成int型数据，然后进行运算。

（2） int → unsigned int →long int → double中表示当运算对象为不同类型数据时的转换方向，如int型数据与unsigned型数据进行运算，int型转换为unsigned型后方可进行运算。int型与double型进行运算，int型直接转换为double型后进行运算，不能理解为先转换为unsigned int型，然后转换为long int型，最后再转换为double型。
